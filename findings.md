# Findings

### Summary of Findings

During this project, I captured and analyzed live network traffic using
Wireshark.

### DNS

-   Observed DNS queries and responses.
-   Learned how domain names are resolved into IP addresses.
-   Identified A, AAAA, HTTPS and CNAME records.

### ICMP / ICMPv6

-   Analyzed connectivity testing using Ping.
-   Observed Echo Request and Echo Reply packets.
-   Observed IPv6 network management messages.

### TCP

-   Studied reliable communication.
-   Observed TCP Three-Way Handshake.
-   Identified ACK, FIN/ACK, Retransmission and ZeroWindow packets.

### TLS

-   Analyzed the TLS handshake.
-   Observed Client Hello, Server Hello, Certificate, Key Exchange,
    Change Cipher Spec and encrypted Application Data.
-   Learned how HTTPS protects communication using encryption.

## Skills Gained

-   Wireshark packet capture
-   DNS analysis
-   ICMP analysis
-   TCP analysis
-   TLS handshake analysis
-   Packet filtering
-   Network troubleshooting
-   Basic network investigation

## Conclusion

This project improved my understanding of network communication and
strengthened the practical skills required for an entry-level SOC
Analyst role.
